# Guia de Planejamento Logístico - Abaré

## 1. Acesso
Você pode acessar o painel de planejamento de duas formas:
- **Menu Lateral**: Vá em **Gestão > Planejamento Abaré**.
- **Página de Beneficiários**: Clique no menu de navegação rápida se disponível ou use o link direto `/planejamento/abare`.

## 2. Como Gerar a Prévia
1. Ao acessar a tela, clique no botão **"Gerar Prévia Inteligente"**.
2. O sistema irá analisar todos os beneficiários do município de **Abaré**.
3. Serão filtrados apenas aqueles que **NÃO** possuem o campo GRH preenchido (excluindo GRH 1, 2, 3, etc.).

## 3. Entendendo os Grupos
O sistema utiliza um algoritmo inteligente para dividir os beneficiários em **3 Eventos Logísticos**:
- **Geolocalização**: Beneficiários próximos geograficamente são agrupados (usando Latitude/Longitude).
- **Comunidade**: Caso não tenham coordenadas, são agrupados pela Comunidade.

## 4. Custos e Cotação
- O painel exibe o **Custo Estimado** baseado no preço médio dos Kits + Logística.
- Para formalizar, clique em **"Baixar PDF"**.
- O documento gerado já vem com a assinatura de **Fabiano - Coordenador Financeiro**.

## 5. Importação de Novos Dados
- Na tela de Beneficiários, use o botão **"Importar Planilha (.xlsx)"** para adicionar novos cadastros ou verificar duplicidades antes de gerar o planejamento.
